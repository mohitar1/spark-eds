(Requires session authentication)
⚙️ Configuration
Your Cloudflare account ID (pre-filled)
Get from: cloudflare/.secrets (KOASSETS_ANALYTICS_API_TOKEN) or Cloudflare Dashboard → My Profile → API Tokens
Analytics Engine dataset name (pre-filled)
Where to find your API token
  1. From project secrets (easiest): Open cloudflare/.secrets and copy the value of KOASSETS_ANALYTICS_API_TOKEN
  2. From Cloudflare Secrets Store: Check cloudflare/wrangler.toml for the ANALYTICS_API_TOKEN binding (secret name: KOASSETS_ANALYTICS_API_TOKEN)
  3. From Cloudflare Dashboard: Go to My Profile → API Tokens → create/use token with "Analytics:Read" permission

Note: Token is stored in browser localStorage only. This tool calls Cloudflare's API directly.

⚠️ CORS Note: Cloudflare blocks browser requests by default. You must either:

  • Run Chrome with CORS disabled (see warning banner if you get errors)
  • Or use the backend proxy option (see README)
